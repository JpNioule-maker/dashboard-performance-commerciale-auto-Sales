# dashboard-performance-commerciale-auto-Sales (2018-2020)


Dashboard Tableau analysant la performance commerciale sur la période 2018-2020, à partir de données de ventes (commandes, clients, lignes de produits, répartition géographique).

## Aperçu

Le dashboard "Performance Commerciale" regroupe :

- **4 indicateurs clés (KPI)** : Chiffre d'affaires total, Nombre de commandes, Quantité commandée, Nombre de clients
- **Répartition géographique des ventes** : carte par pays + classement Top 10 pays
- **Évolution du chiffre d'affaires** : vue annuelle et vue mensuelle
- **Analyse par ligne de produit** : chiffre d'affaires par catégorie + ventes mensuelles par catégorie
- **Saisonnalité** : répartition des ventes par trimestre
